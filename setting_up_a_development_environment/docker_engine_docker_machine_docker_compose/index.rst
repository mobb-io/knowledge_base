Docker, Docker-Machine and Docker-Compose
=========================================

`Docker`_ is a containerization tool used for spinning up isolated, reproducible application environments. This piece details how to containerize a `Django Project`_, `Postgres`_, and `Redis`_ for local development along with delivering the stack to the cloud via `Docker-Compose`_ and `Docker-Machine`_.
